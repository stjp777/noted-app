# Noted

First beginner project
A full-stack productivity web app built with Python and Flask.

## Features
- **Notes** - add and delete personal notes
- **Reminders** - checklist with toggle and delete
- **Schedule** - weekly schedule timetable organized by day

## Tech Stack
- Python, Flask, SQLAlchemy, SQLite
- HTML, CSS, Jinja1
- Deployed on Render
